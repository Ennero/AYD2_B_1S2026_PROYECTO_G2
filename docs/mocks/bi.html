Mapa API del Dashboard BI

Cada vista gerencial se alimenta con un endpoint distinto, no con una sola consulta gigante.

GET/api/bi/kpis/daily

Tarjetas superiores

Llena `completedServices`, `dailyBilling` y `activeIncidents` del corte diario.

GET/api/bi/branches/distribution

Operaciones por sede

Alimenta el gráfico de distribución y la comparación de volumen por sucursal.

GET/api/bi/profitability

Rentabilidad y gasto crítico

Entrega margen promedio, ingreso vs gasto y el rubro más sensible para gerencia.

GET/api/bi/orders/recent

Últimos despachos

Llena la tabla de monitoreo rápido con las órdenes más recientes, su cliente, ruta y estado.
